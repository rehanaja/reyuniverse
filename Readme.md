Judul Proyek: Website Portfolio Pribadi

Deskripsi Singkat:

Proyek ini merupakan sebuah website portfolio pribadi yang dirancang dan dikembangkan sebagai bagian dari tugas Ujian Tengah Semester (UTS) mata kuliah Pemrograman Web. Website ini bertujuan untuk menampilkan informasi tentang diri saya, keahlian, serta beberapa contoh karya atau proyek yang telah saya buat.

Website dibangun dengan menggunakan teknologi dasar seperti HTML, CSS, dan JavaScript, dengan struktur yang responsif agar dapat diakses baik melalui perangkat desktop maupun mobile.

Fitur-fitur utama:

Halaman Beranda: Menampilkan nama, foto, dan deskripsi singkat sebagai pengantar.

Tentang Saya: Berisi informasi pribadi, latar belakang pendidikan, dan keahlian.

Portofolio: Menampilkan daftar proyek/karya dengan gambar dan deskripsi.

Kontak: Formulir kontak sederhana untuk pengunjung yang ingin menghubungi.

Navigasi Responsive: Navigasi bar yang dapat diakses dengan baik di berbagai ukuran layar.

Tujuan Pembelajaran:

Menerapkan konsep dasar HTML, CSS, dan JavaScript dalam membangun website statis.

Memahami struktur layout dan prinsip desain web yang responsif.

Mengorganisir file proyek dengan struktur direktori yang baik.

Hasil Akhir:

Judul Proyek: Website Portfolio Pribadi

Deskripsi Singkat:

Proyek ini merupakan sebuah website portfolio pribadi yang dirancang dan dikembangkan sebagai bagian dari tugas Ujian Tengah Semester (UTS) mata kuliah Pemrograman Web. Website ini dibuat untuk menampilkan informasi diri, keahlian, serta contoh-contoh proyek yang pernah saya kerjakan, dan juga menyertakan penggunaan bahasa pemrograman PHP untuk memproses data secara dinamis di sisi server.

Website dibangun dengan menggunakan HTML, CSS, JavaScript, dan PHP, serta memiliki struktur yang responsif agar dapat diakses melalui berbagai perangkat.

Fitur-fitur utama:

Halaman Beranda: Menampilkan nama, foto, dan deskripsi singkat.

Tentang Saya: Berisi informasi pribadi dan latar belakang pendidikan.

Portofolio: Menampilkan daftar proyek dalam bentuk galeri.

Kontak: Tersedia formulir kontak yang diproses menggunakan PHP untuk mengirimkan data ke email atau menyimpan ke file/DB.

Navigasi Responsive: Menggunakan JavaScript dan CSS agar bisa digunakan di mobile/desktop.

Penerapan PHP:

Form Kontak: Data formulir dikirim menggunakan metode POST dan diproses dengan script PHP (contact.php) untuk validasi dan penanganan data.

Template Include: Menggunakan include() atau require() untuk mengatur bagian header, footer, dan navbar agar lebih modular.

(Opsional) Menyimpan data pengunjung/form ke file .txt atau database sederhana seperti MySQL.

Tujuan Pembelajaran:

Mengimplementasikan struktur layout responsif.

Menerapkan dasar pemrograman server-side menggunakan PHP.

Mempelajari pengolahan form dan modularisasi file PHP.

Memahami arsitektur proyek web dinamis tingkat dasar.

Hasil Akhir:
Website ini diharapkan dapat menjadi portofolio online sekaligus sebagai media latihan dalam mengintegrasikan teknologi frontend dan backend sederhana menggunakan PHP.
